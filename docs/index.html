Build Beautiful Terminal UIs in Swift

TUIKit is a modern Swift framework for creating sophisticated, interactive terminal user interfaces on macOS and Linux with a declarative, SwiftUI-like API.

Get Started View on GitHub 
import TUIKit

@main
struct MyApp: App {
    var body: some Scene {
        WindowGroup {
            VStack(spacing: 1) {
                Text("Welcome!")
                    .bold()
                    .foregroundColor(
                        .theme.accent
                    )
            }
            .padding()
        }
    }
}

No Dependencies

TUIKit is a pure Swift implementation with zero external dependencies. It works perfectly on macOS 10.15+ and Linux with glibc.

  • Swift 6.0 or later
  • macOS 10.15+ support
  • Full Linux compatibility
  • Terminal with ANSI 256 color support
  • Zero external dependencies
